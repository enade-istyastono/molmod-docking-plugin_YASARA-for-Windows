# MOLMOD Docking Plugins for YASARA Structure for Windows YASARA Structure plugins for performing protein-ligand docking simulations, developed by the CTO of MOLMOD ID.

Dependencies:
1. YASARA Structure for Windows (version minimum: 25.1.13)
2. Ubuntu (tested working version: 24.04) in Windows Subsystem for Linux.

Installation: Copy all mcr and sh files to the folder {YASARADir}\plg or the plugin folder of YASARA Structure.

Note: Tutorial to use the plugins is under preparation.
