# MOLMOD Docking Plugins for YASARA Structure for Unix YASARA Structure plugins for performing protein-ligand docking simulations. An improved version from the 2023 version.

Developed and maintained by:
Enade P. Istyastono
Department of Pharmacy, School of Medicine and Health Sciences, Atma Jaya Catholic University of Indonesia.

Dependencies:
YASARA Structure for Unix (version minimum: 25.1.13)

Installation: Open YASARA Structure and run the install.mcr file from the software. To uninstall the plugins, run the uninstall.mcr file from the software.

Note: Tutorial to use the plugins is under preparation.
